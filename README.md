# Sistema-Corporativo-Gestion-Empleados-Impresistem
Este repositorio almacenara el proyecto para la gestión de empleados y otros componentes empresariales, integrados mediante un API Gateway y comunicándose entre sí a través de servicios REST. Además, la arquitectura debe proveer una API de información de clientes que pueda ser consumida por otras plataformas internas de la empresa
